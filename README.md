University Event Manager

How to set up:
1. clone the repository
2. install dependancies
    2.1 pip install flask flask-restful flask-sqlalchemy google-auth-oauthlib google-api-python-client
3. incase .db files aren't present, run 'python prepopulate.py'
4. run it - 'python app.py'
5. open browser
